# Apa yang ada di GitHub vs lokal

Repo publik **tidak** memuat strategi hackathon atau prompt internal.

| Hanya lokal (`.gitignore`) | Di GitHub |
|----------------------------|-----------|
| `docs/` — pitch, timeline, bisnis, arsitektur detail | `README.md` — narasi publik |
| `AGENTS.md` — persona QA & verifikasi | Kode aplikasi (saat ada) |
| `prompts/` — system prompt produksi | `.env.example` (placeholder kosong) |
| `agents/` — template laporan audit | `.cursor/rules`, skrip `scripts/` |

Clone untuk develop: simpan folder `docs/` dan `AGENTS.md` dari cadangan tim atau salinan pribadi Anda.
