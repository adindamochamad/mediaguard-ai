# Brand assets (served from `public/brand/`)

Production files are copied from `asset/` at repo root. Do not hotlink from `asset/` in the app.

| File | Usage |
|------|--------|
| `logo-header.png` | Header (40×40) |
| `logo-master.png` | Full logo source |
| `hero-background.png` | Hero `Image` fill |
| `og-social.png` | Open Graph / Twitter |
| `empty-state.png` | Roadmap section |
| `icon-512.png`, `icon-192.png`, `favicon-32.png`, `apple-touch-icon.png` | PWA / browsers |

Next.js also uses `app/icon.png` and `app/apple-icon.png` (generated from icon source).
